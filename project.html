BioData Form

This project is a simple HTML form that collects basic biodata from users. It includes fields such as name, contact number, email, address, age, date of birth, section, year level, and hobbies.

This form is designed to be user-friendly and easy to fill out. The information is later displayed using JavaScript for real-time updates.

JavaScript Array Display

This project is a simple JavaScript program that displays information from an array containing details like name, contact number, email, address, age, date of birth, section, year level, and hobbies. The program dynamically updates the webpage to display the user's information.

Average of 4 Numbers

This JavaScript program allows the user to input four numbers. It calculates and displays the average of these numbers.
